## UI

`@tszhong0411/ui` is a modern component library built with [shadcn/ui](https://github.com/shadcn-ui/ui). It provides a collection of reusable, accessible and customizable components to streamline your web development workflow.

### Installation

```bash
pnpm add @tszhong0411/ui
```

### Usage

```css
@import 'tailwindcss';
@import '@tszhong0411/ui/styles/preset.css';
@import '@tszhong0411/ui/styles/shiki.css'; /* Optional */
@import '@tszhong0411/ui/styles/typography.css'; /* Optional */
@source '../../node_modules/@tszhong0411/ui/dist/**/*.js';
```

```tsx
import { Button } from '@tszhong0411/ui'
```

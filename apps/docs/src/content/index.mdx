## UI

`@isyuricunha/ui` is a modern component library built with [shadcn/ui](https://github.com/shadcn-ui/ui). It provides a collection of reusable, accessible and customizable components to streamline your web development workflow.

### Installation

```bash
pnpm add @isyuricunha/ui
```

### Usage

```css
@import 'tailwindcss';
@import '@isyuricunha/ui/styles/preset.css';
@import '@isyuricunha/ui/styles/shiki.css'; /* Optional */
@import '@isyuricunha/ui/styles/typography.css'; /* Optional */
@source '../../node_modules/@isyuricunha/ui/dist/**/*.js';
```

```tsx
import { Button } from '@isyuricunha/ui'
```

## Usage

```tsx
import { Collapsible, CollapsibleContent, CollapsibleTrigger } from '@tszhong0411/ui'
```

```tsx
<Collapsible>
  <CollapsibleTrigger>What is Next.js?</CollapsibleTrigger>
  <CollapsibleContent>
    Next.js is a React framework that enables functionality such as server-side rendering and
    generating static websites for React based web applications.
  </CollapsibleContent>
</Collapsible>
```

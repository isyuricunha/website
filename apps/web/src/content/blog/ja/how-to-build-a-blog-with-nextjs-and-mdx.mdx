## ブログの作成方法

まず、以下のコマンドでNext.jsプロジェクトを作成します：

```bash title='Terminal'
yarn create next-app nextjs-mdx-blog
```

次に、以下のファイルを作成します：

- `components/layout.jsx` - すべてのコンポーネントをコンテナで包む（オプション、スタイル用）
- `data/blog/*.mdx` - ブログ記事
- `lib/format-date.js` - 日付を`MM DD, YYYY`形式でフォーマット
- `pages/blog/[slug].jsx` - 記事ページ、[動的ルート](https://nextjs.org/docs/routing/dynamic-routes)を使用

### Markdownファイルの処理方法

まず`const root{:js}` - ルートディレクトリ、そして`process.cwd(){:js}`メソッドはNode.jsプロセスの現在の作業ディレクトリを返します。

```js title='lib/mdx.js'
const root = process.cwd()
```

もう一つの変数`POSTS_PATH{:.entity.name.constant}`は記事ファイルが保存されているパスです。

```js title='lib/mdx.js'
import path from 'path'

const POSTS_PATH = path.join(root, 'data', 'blog') // [!code highlight]
// 出力: A:/nextjs-mdx-blog/data/blog
```

次に、fsを使用してそのディレクトリの内容、つまり`data/blog`下のすべてのファイル名を読み取ります。

```js title='lib/mdx.js'
import fs from 'fs'

export const allSlugs = fs.readdirSync(POSTS_PATH)
// 出力: ['markdown.mdx', 'nextjs.mdx', 'react.mdx']
```

次に、ファイル拡張子を削除する関数を書きます。これは後で使用されます。

```js title='lib/mdx.js'
export const formatSlug = (slug) => slug.replace(/\.mdx$/, '')
/**
 * 例: formatSlug('markdown.mdx')
 * 出力: 'markdown'
 */
```

次のステップは、スラッグによって記事の内容を取得することです。

```js title='lib/mdx.js'
export const getPostBySlug = async (slug) => {
  const postFilePath = path.join(POSTS_PATH, `${slug}.mdx`)
  // 出力: A:/nextjs-mdx-blog/data/blog/slug.mdx
  const source = fs.readFileSync(postFilePath)
  // ファイルの内容を返す

  const { content, data } = matter(source)
  /*
   * 例:
   *  ---
   *  title: Hello
   *  slug: home
   *  ---
   *  <h1>Hello world!</h1>
   *
   * 戻り値:
   *  {
   *    content: '<h1>Hello world!</h1>',
   *    data: {
   *      title: 'Hello',
   *      slug: 'home'
   *    }
   *  }
   */

  const mdxSource = await serialize(content)
  // コンテンツをserialize（next-mdx-remote）に渡して処理

  const frontMatter = {
    ...data,
    slug
  }
  // スラッグもフロントマターに入れる、これは後で使用される

  return {
    source: mdxSource,
    frontMatter
  }
}
```

次に、ホームページに表示するすべての記事を取得できます。

```js title='lib/mdx.js'
export const getAllPosts = () => {
  const frontMatter = []

  allSlugs.forEach((slug) => {
    const source = fs.readFileSync(path.join(POSTS_PATH, slug), 'utf-8')

    const { data } = matter(source)

    frontMatter.push({
      ...data,
      slug: formatSlug(slug),
      date: new Date(data.date).toISOString()
    })
  })

  return frontMatter.sort((a, b) => dateSortDesc(a.date, b.date))
}

// 日付の降順でソート
const dateSortDesc = (a, b) => {
  if (a > b) return -1
  if (a < b) return 1

  return 0
}
```

### 日付のフォーマット

```js title='lib/format-date.js'
export const formatDate = (date) =>
  new Date(date).toLocaleDateString('en', {
    year: 'numeric',
    month: 'long',
    day: 'numeric'
  })
/*
 * formatDate('2022-08-21T00:00:00Z')
 * 出力: 'August 21, 2022'
 */
```

### ホームページ

```jsx title='pages/index.jsx'
import { formatDate } from '../lib/format-date'
import { getAllPosts } from '../lib/mdx'

import Link from 'next/link'

export default function Home({ posts }) {
  return (
    <>
      <h1 className='mb-8 text-6xl font-bold'>ブログ</h1>
      <hr className='my-8' />
      <ul className='flex flex-col gap-3'>
        {posts.map(({ slug, title, summary, date }) => (
          <li key={slug}>
            <Link href={`/blog/${slug}`}>
              <a className='block rounded-lg border border-solid border-gray-300 p-6 shadow-md'>
                <div className='flex justify-between'>
                  <h2>{title}</h2>
                  <time dateTime={date}>{formatDate(date)}</time>
                </div>
                <p className='mt-4'>{summary}</p>
              </a>
            </Link>
          </li>
        ))}
      </ul>
    </>
  )
}

// getStaticPropsを使用してすべての記事を取得
export const getStaticProps = async () => {
  const posts = getAllPosts()

  return {
    props: {
      posts
    }
  }
}
```

### 記事ページ

```jsx title='pages/blog/[slug].jsx'
import { formatDate } from '../../lib/format-date'
import { allSlugs, formatSlug, getPostBySlug } from '../../lib/mdx'

import { MDXRemote } from 'next-mdx-remote'

export default function Blog({ post }) {
  const { title, date } = post.frontMatter

  return (
    <div>
      <h1 className='mb-2 text-6xl font-bold'>{title}</h1>
      <time dateTime={date} className='text-lg font-medium'>
        {formatDate(date)}
      </time>
      <hr className='my-8' />
      <article className='prose max-w-none'>
        <MDXRemote {...post.source} />
      </article>
    </div>
  )
}

export const getStaticProps = async ({ params }) => {
  const post = await getPostBySlug(params.slug)

  return {
    props: {
      post
    }
  }
}

export const getStaticPaths = async () => {
  const paths = allSlugs.map((slug) => ({
    params: {
      slug: formatSlug(slug)
    }
  }))
  /*
   * paths出力:
   *   [
   *     { params: { slug: 'markdown' } },
   *     { params: { slug: 'nextjs' } },
   *     { params: { slug: 'react' } }
   *   ]
   */

  return {
    paths,
    fallback: false
  }
}
```

このようにして、シンプルなブログが完成します。

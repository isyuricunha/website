## PHPの紹介

PHP（正式名称：PHP: _Hypertext Preprocessor_）は、オープンソースの汎用コンピュータスクリプト言語で、特にWeb開発に適しており、HTMLに埋め込むことができます。PHPの構文は、C言語、Java、Perlなどの人気のあるコンピュータ言語の特徴を取り入れており、一般的なプログラマーが学習しやすくなっています。PHPの主な目標は、Web開発者が動的なページを素早く書けるようにすることですが、PHPは他の多くの分野でも使用されています。

PHPファイルの拡張子は`.php`です。例：`index.php`。上記のように、`PHP`は`HTML`に直接埋め込むことができます。

## 使用方法

ブラウザが上から下に`<?php`を読み取ると、`PHP`を実行し、`?>`で終了します。例：

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "これは文章です。"
        ?>
    </body>
</html>
```

## 注意事項

`PHP`では、`PHP`ディレクティブが完了した後に他の`HTML`がない場合、`?>`で終了しないことをお勧めします。閉じると問題が発生します。

```php
<?php
    echo "PHP終了後にHTMLなし";
```

## PHPコメント

ほとんどのWebプログラミング言語はコメントできます。`PHP`の**単一行**の方法は`//`で、**複数行**の方法は`/* ここにテキスト */`です。

```php
<?php
    echo = "これは単一行コメントです"; // 私は単一行コメントです
    echo = "これは複数行コメントです";
    /*
        私は複数行コメントです
        2行目
    */
?>
```

## Echo

`PHP`で最もシンプルな構文は`Echo`です。`Echo`の内容は二重引用符`""`または単一引用符`''`で囲む必要があります。各構文はセミコロン`;`で終了する必要があります。

```php
<?php
    echo "これは二重引用符を使用した例です。";
    echo 'これは単一引用符を使用した例です。';
?>
```

`Echo`の内容は`HTML`構文も使用できます。例えば、ここで`<br/>`を使用したので、改行できます。

```php
<?php
    echo "これは最初の文章です<br/>";
    echo "これは2番目の文章です";
    echo "<a href=\"https://example.com\">ここはリンクです</a>";
?>
```

## 変数

`Javascript`と同様に、変数に値を割り当てて`PHP`を書きやすくすることができます。ドル記号`$`を使用します。

```php
<?php
    $value = "私は値です";
?>
```

変数に名前を付けるときに従うべきルールがいくつかあります。

<Table
  headers={['例', '説明', 'ルール']}
  rows={[
    ['$value', '小文字', '✅'],
    ['$Value', '大文字', '✅'],
    ['$_value', 'アンダースコア', '✅'],
    ['$1value', '数字から開始', '❌']
  ]}
/>

変数名では`大文字`と`小文字`に違いがあります。大文字の変数は小文字の変数と等しくありません。

## 変数命名のヒント

### キャメルケース

変数名と関数名が2つ以上の単語によってリンクされて一意の識別語を形成する場合、「キャメルケース」を使用して表現すると、変数と関数の可読性を高めることができます。変数に`スペース`を入れることができないためです。

```php
<?php
    $onePlusOneAnswer = 1 + 1;
    echo $onePlusOneAnswer;
?>
```

### スネークケース

スネークケース（snake_caseのような）は、すべてのスペースがアンダースコア（\_）文字に置き換えられ、すべての単語の最初の文字が小文字で書かれる書き方スタイルを指します。

```php
<?php
    $one_plus_one_answer = 1 + 1;
    echo $one_plus_one_answer;
?>
```

## 文字列演算子

2つの文字列演算子があります。1つ目は連結演算子（"."）で、左右の引数の連結された文字列を返します。2つ目は連結代入演算子（".="）で、右側の引数を左側の引数に追加します。

```php
<?php
    echo "これ" . " " . "は" . " " . "文字列" . " " . "演算子" . "です。";
?>
```

## 算術演算子

```php
<?php
    $value0 = 1.5 // Float
    $value1 = 1 + 1 // 加算
    $value2 = 1 - 1 // 減算
    $value3 = 1 * 1 // 乗算
    $value4 = 1 / 1 // 除算
    $value5 = 12 / 5 // 除算 | 不完全な除算はfloatになります。これは他のプログラミング言語とは異なり、phpは余りを計算します
    $value6 = 3 % 5 // 余りを取る
?>
```

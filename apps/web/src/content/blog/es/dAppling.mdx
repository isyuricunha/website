# Introducción

La red descentralizada está creciendo rápidamente, pero uno de los mayores desafíos que enfrentan los desarrolladores es encontrar una manera de organizar sus frontends de una manera que sea resistente a la censura y segura. Dappling Network es una nueva plataforma que tiene como objetivo resolver este problema al proporcionar una forma simple y fácil de usar para alojar frontends descentralizados.

# ¿Qué es la red Dappling?

Dappling Network es un servicio de alojamiento web creado específicamente para los protocolos Web3. Utiliza el sistema de archivos interplanetarios (IPFS) para almacenar y servir código frontend, lo que lo hace resistente a la censura y descentralizado. Dappling también proporciona una serie de otras características que lo convierten en una herramienta valiosa para los desarrolladores de Web3, que incluyen:

- ** Implementaciones verificables: ** La mancha utiliza un proceso de implementación verificable que garantiza que su código frontend se implemente correctamente y que no pueda ser manipulado.
- ** Frontends controlados por contratos inteligentes: ** La manchas le permite controlar sus frontends usando contratos inteligentes, lo que le brinda aún más control sobre sus aplicaciones.
- ** Rendimiento: ** Dappling utiliza IPFS para servir su código frontend, lo que lo hace muy desempeñado.

# ¿Cómo funciona la mancha?

Para usar la mancha, simplemente necesita crear una cuenta y cargar su código frontend. Dappling implementará su código en IPFS y generará una URL que puede usar para acceder a su frontend.

# ¿Por qué utilizar la red de manchas?

Hay varias razones por las que debe usar Dappling Network para alojar sus frontends, que incluyen:

- ** Resistencia a la censura: ** La manchas es resistente a la censura porque usa IPFS para almacenar y servir su código frontend. Esto significa que su interfaz no puede ser censurado por gobiernos u otras organizaciones.
- ** Seguridad: ** Dappling utiliza una serie de características de seguridad para proteger sus frontends, incluidas las implementaciones verificables y las frontends inteligentes controladas por contratos.
- ** Rendimiento: ** Dappling utiliza IPFS para servir su código frontend, lo que lo hace muy desempeñado.
- ** Facilidad de uso: ** La mancha es muy fácil de usar. Puede crear una cuenta e implementar su interfaz en solo unos minutos.

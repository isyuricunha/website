# Introducción

La web descentralizada está creciendo rápidamente, pero uno de los mayores desafíos para los desarrolladores es encontrar una manera de alojar sus frontends de forma resistente a la censura y segura. **dAppling Network** es una nueva plataforma que busca resolver este problema al ofrecer una forma sencilla y fácil de usar para alojar frontends descentralizados.

---

# ¿Qué es dAppling Network?

**dAppling Network** es un servicio de alojamiento web diseñado específicamente para protocolos Web3. Utiliza el **Sistema de Archivos InterPlanetario (IPFS)** para almacenar y servir el código del frontend, lo que lo hace resistente a la censura y descentralizado. Además, dAppling ofrece una serie de características adicionales que lo convierten en una herramienta valiosa para los desarrolladores de Web3, como:

- **Implementaciones verificables:** dAppling utiliza un proceso de implementación verificable que garantiza que el código de tu frontend se implemente correctamente y que no pueda ser alterado.
- **Frontends controlados por contratos inteligentes:** dAppling te permite controlar tus frontends mediante contratos inteligentes, lo que te brinda un mayor control sobre tus aplicaciones.
- **Rendimiento:** dAppling utiliza IPFS para servir el código de tu frontend, lo que lo hace muy eficiente.

---

# ¿Cómo funciona dAppling?

Para usar dAppling, simplemente necesitas crear una cuenta y subir el código de tu frontend. dAppling implementará tu código en IPFS y generará una URL que podrás utilizar para acceder a tu frontend.

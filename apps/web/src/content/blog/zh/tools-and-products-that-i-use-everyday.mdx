工具不断变化，但有些东西却始终如一。在下面的列表中，我整理了所有在工作和生活中使用的工具。如果你对列表中的任何内容有疑问，请随时[联系我](mailto:contact@yuricunha.com)。

## 效率工具

**[Alfred](https://www.alfredapp.com/)**  

增强版 Spotlight。我每天至少使用 Alfred 20 次来打开应用程序、访问剪贴板和执行小型工作流（例如刷新 Wi-Fi）。我毫不犹豫地购买了 Powerpack，并强烈推荐给所有 Mac 用户。

**[Dynalist](https://dynalist.io/)**  

Dynalist 让我可以组织所有想法。它类似于可无限扩展的项目符号列表，每个项目符号都可以作为单独的页面。我将其用作笔记应用、偶尔的日记本以及写作工具。

**[Tripmode](https://www.tripmode.ch/)**  

使用热点时，我总是担心消耗过多数据流量。有了 Tripmode，我可以轻松管理哪些应用程序可以访问互联网，并查看每个会话的流量消耗。

**[Rescuetime](https://www.rescuetime.com/)**  

Rescuetime 在我的 Mac 和 Android 设备后台运行，跟踪屏幕使用时间。我通常不敢查看统计数据。日历集成功能非常适合快速了解总体情况。

**[LastPass](https://www.lastpass.com/)**  

LastPass 是最大的时间节省工具。我在个人和工作中使用它来管理密码和存储机密数据（例如社会保障号码）。没有它，我会感到无所适从。

## 设计工具

**[SIP](https://sipapp.io/)**  

SIP 是一款出色的取色工具。我主要使用取色功能，但其他功能也非常不错。我设置的快捷键是 `⌥ + SHIFT + -`。

**[Framer](https://www.framer.com/)**  

Framer 让我可以在一个项目中进行设计和编码，因此原型可以使用真实数据、交互，并且更加精细。我非常喜欢它，并尽可能使用 Framer。唯一的缺点是目前缺少网页版。

**[Figma](https://www.figma.com/)**  

Figma 是协作设计的行业标准。它直观易用，拥有丰富的插件生态系统，并且性能出色。我在与其他设计师或开发人员合作时使用 Figma。

**[Adobe Creative Cloud](https://www.adobe.com/creativecloud.html)**  

我和女友共享 Adobe Creative Cloud 订阅。我只使用 Lightroom、Photoshop 和 Illustrator。我认为大多数 Adobe 应用程序过于复杂，功能强大但容易让人沮丧。

**[Canva](https://www.canva.com/)**  

Canva 是一款简单但功能丰富的在线设计工具，拥有大量模板，适用于各种格式。我用它来制作社交媒体帖子、印刷品设计等。

## 开发工具

**[VS Code](https://code.visualstudio.com/)**  

自 2018 年末开始编程以来，VS Code 迅速流行。它速度快，社区提供了丰富的扩展。我所有的代码都在 VS Code 中编写，从未遇到过问题。

**[Netlify](https://www.netlify.com/)**  

我对托管和后端不太熟悉，所以我非常喜欢 Netlify。它简单易用，让我专注于自己擅长的事情。我使用 Netlify 的免费计划来托管静态项目，还收到了一双 Netlify 赠送的袜子。

**[Namecheap](https://www.namecheap.com/)**  

我的第一个域名是在 Namecheap 购买的，从未更换过。可能有更好的域名注册商，但我一直没有勇气更换。如果你觉得我应该换，请告诉我。

**[GitHub](https://github.com/)**  

我的所有代码都托管在 GitHub 上。你可以[在这里](https://github.com/wirtzdan/new-website)查看本网站的代码。如果不明白，可以联系我，我很乐意帮助。

**[iTerm 2](https://iterm2.com/)**  

我使用 iTerm 作为终端客户端，搭配 zsh。说实话，我不知道为什么要用它，但它的主题很好看。

**[ImageOptim](https://imageoptim.com/mac)**  

ImageOptim 是一个隐藏的宝藏。这款小工具可以一次性压缩 100 张图片，并且通常能减少一半的文件大小。我搭配 macOS 预览工具来裁剪、缩放和压缩图片，以适应网络需求。

## 电子设备

**[Peak Design Everyday Backpack](https://www.peakdesign.com/products/everyday-backpack)**  

Peak Design 是我非常信赖的品牌。我最初对这款背包犹豫不决，但最终发现它物超所值。我每天都在使用它，无论是工作、旅行、购物还是摄影。

**[MacBook Pro (15 英寸, 2017)](https://www.apple.com/shop/buy-mac/macbook-pro)**  

我的移动工作站，也是我背包的固定搭档。我每天花大量时间使用它，尽管价格昂贵，但非常值得投资。

**[富士 X-T10](https://www.fujifilmusa.com/products/digital_cameras/x/fujifilm_x_t10/)**  

这台相机对我来说意义非凡，我在大学时买的。它小巧、操作顺畅，成像质量令人惊艳。我常用两款定焦镜头（23mm F2 & 50mm F2）。

**[Magic Mouse 2](https://www.apple.com/de/shop/product/MLA02Z/A/magic-mouse-2-silber)**  

我在 eBay 上以 30 欧元买到的 Apple Magic Mouse 2，在任何表面上都能顺畅使用。唯一的问题是它需要电池，而我经常忘记带备用电池。

**[人体工学笔记本支架](https://www.amazon.de/dp/B01N99IGMQ/ref=sr_1_5?crid=2XJA82MXCVRRC&keywords=mobiler+laptop+st%C3%A4nder&qid=1573384409&sprefix=mobile+laptop%2Caps%2C200&sr=8-5)**  

这是我找到的最小、最轻便的笔记本支架，随时挂在背包侧面，展开速度极快。

**[罗技 K380](https://www.logitech.com/en-us/product/multi-device-keyboard-k380)**  

这个键盘[完美适配](https://twitter.com/wirtzdan/status/1193482720615243776?s=19)我的背包上层空间，这种契合感令人满意。搭配笔记本支架、鼠标和背包，我可以随时随地搭建灵活的工作站。

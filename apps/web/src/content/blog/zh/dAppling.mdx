# 引言

去中心化网络正在迅速发展，但开发者面临的最大挑战之一是如何托管前端，同时确保其抗审查性与安全性。dAppling Network 正是为解决这一问题而诞生，它提供了一种简单易用的方法来托管去中心化前端。

# 什么是 dAppling Network？

dAppling Network 是一项专为 Web3 协议打造的网络托管服务。它采用星际文件系统 (IPFS) 来存储和提供前端代码，这使得前端具备抗审查性和去中心化的特点。dAppling 还提供了许多其他功能，为 Web3 开发者增添了额外价值，包括：

- **可验证部署：** dAppling 采用可验证的部署流程，确保你的前端代码正确部署且无法被篡改。
- **智能合约控制的前端：** dAppling 允许你通过智能合约来控制前端，为你的应用提供更多控制权。
- **性能：** 通过 IPFS 提供前端代码，使其运行速度快且稳定。

# dAppling 如何工作？

使用 dAppling 非常简单。你只需创建一个账户并上传你的前端代码，dAppling 将自动把代码部署到 IPFS 上，并生成一个 URL，供你访问前端。

选择 dAppling Network 托管前端有以下诸多理由：

- **抗审查性：** 由于 dAppling 使用 IPFS 存储和提供前端代码，因此你的前端不会受到政府或其他组织的审查。

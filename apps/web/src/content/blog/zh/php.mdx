## PHP 简介

PHP（全名：PHP: _Hypertext Preprocessor_）是一种开源的通用计算机脚本语言，特别适合用于网页开发并嵌入到 HTML 中。PHP 的语法借鉴了 C 语言、Java 和 Perl 等流行计算机语言的特点，对于一般程序员来说易于学习。PHP 的主要目标是让网页开发人员快速编写动态页面，但 PHP 也用于许多其他领域。

PHP 文件的后缀名为 `.php`，例如 `index.php`。如上所述，`PHP` 可以直接嵌入到 `HTML` 中。

## 用法

当浏览器从上到下读取 `<?php` 时，会执行 `PHP`，并以 `?>` 结束。例如：

```php
<!DOCTYPE html>
<html>
    <head>
        <title>PHP</title>
    </head>
    <body>
        <?php
            echo "这是一个句子。"
        ?>
    </body>
</html>
```

## 注意事项

在 `PHP` 中，`PHP` 指令结束后不应该有其他 `HTML`。建议不要以 `?>` 结束，否则关闭时可能会引起问题。

```php
<?php
    echo "PHP 结束后没有 HTML";
```

## PHP 注解

几乎所有的网页程序语言都可以使用注解，`PHP` 的**单行注解**使用 `//`，**多行注解**使用 `/* Some text here */`。

```php
<?php
    echo = "这是一个单行注解"; // 我是一个单行注解
    echo = "这是一个多行注解";
    /*
        我是一个多行注解
        第二行
    */
?>
```

## Echo

`PHP` 中最简单的语法是 `Echo`。`Echo` 的内容必须用双引号 `""` 或单引号 `''` 包起来。每条语法必须以分号 `;` 结束。

```php
<?php
    echo "这是使用双引号的示例。";
    echo '这是使用单引号的示例。';
?>
```

在 `Echo` 的内容中还可以使用 `HTML` 语法。例如，使用 `<br/>` 来换行。

```php
<?php
    echo "这是第一句<br/>";
    echo "这是第二句";
    echo "<a href=\"https://example.com\">这是一个连结</a>";
?>
```

## 变数

我们可以将值分配给变数，使我们更容易编写 `PHP`，类似于 `Javascript`。我们将使用美元符号 `$`。

```php
<?php
    $value = "我是变数值";
?>
```

命名变数时需要遵循一些规则。

变数名称中的大写和小写是有区别的。大写变数不等于小写变数。

## 变数命名技巧

### 驼峰式命名法

当变数名称和函数名称由两个或多个单词组合而成时，使用 "驼峰式命名法" 可以提高变数和函数的可读性，因为变数中不能有空格。

```php
<?php
    $onePlusOneAnswer = 1 + 1;
    echo $onePlusOneAnswer;
?>
```

### 蛇形命名法

蛇形命名法（如 snake_case）是指每个空格用下划线（ \_ ）字元替代，并且每个单词的首字母用小写。

```php
<?php
    $one_plus_one_answer = 1 + 1;
    echo $one_plus_one_answer;
?>
```

## 字串运算符

有两种字串运算符。第一种是连接运算符（"."），它返回左右参数连接后的字串。第二种是连接赋值运算符（".="），它将右侧参数附加到左侧参数。

```php
<?php
    echo "这是" . " " . "字串" . " " . "运算符" . "。";
?>
```

## 算术运算符

```php
<?php
    $value0 = 1.5 // 浮点数
    $value1 = 1 + 1 // 加
    $value2 = 1 - 1 // 减
    $value3 = 1 * 1 // 乘
    $value4 = 1 / 1 // 除
    $value5 = 12 / 5 // 除法 | 不完整的除法将变为浮点数，这与其他程序语言不同，PHP 会计算余数
    $value6 = 3 % 5 // 取余数
?>
```

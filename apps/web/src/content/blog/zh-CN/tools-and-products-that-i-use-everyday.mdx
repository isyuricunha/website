工具来来去去，但有些东西却经久不衰。下面的列表中，我整理了所有我在工作和生活中使用的工具。如果你对列表中的任何内容有疑问，请随时[给我发消息](mailto:contact@yuricunha.com)。

## 生产力

**[Alfred](https://www.alfredapp.com/)**

超级强大的 Spotlight。我每天至少使用 Alfred 20 次，用它来打开应用程序、访问剪贴板，以及启动小型工作流（例如刷新我的 Wi-Fi）。我很乐意购买了 Powerpack，并向所有使用 Mac 的人推荐 Alfred。

**[Dynalist](https://dynalist.io/)**

Dynalist 让你可以记录脑海中闪过的一切想法。想象一下，你可以无限深入地创建项目符号列表，并且任何项目符号都可以被视为一个独立的页面。我将其用作我的笔记应用程序、偶尔的日记以及写作工具。

**[Tripmode](https://www.tripmode.ch/)**

我以前总是害怕在使用热点时消耗太多流量。现在不再有这种担忧了。使用 Tripmode，我可以轻松管理哪些应用程序可以访问互联网，并查看在一次会话中使用了多少流量。

**[Rescuetime](https://www.rescuetime.com/)**

Rescuetime 在我的 Mac 和 Android 设备上后台运行，并跟踪我的屏幕使用时间。我经常不敢查看统计数据。日历集成功能非常适合快速查看概况。

**[LastPass](https://www.lastpass.com/)**

LastPass 是最大的时间节省工具。我个人和工作中都使用它来管理密码和存储机密数据（例如社保号码）。没有这个应用程序，我会完全迷失方向。

## 设计

**[SIP](https://sipapp.io/)**

SIP 是一个漂亮的颜色选择器。我只使用颜色选择器功能，但应用程序的其他部分也非常不错。我的自定义快捷键是 `⌥ + SHIFT + -`。

**[Framer](https://www.framer.com/)**

Framer 让你可以在一个项目中同时进行设计和编码。这意味着原型可以使用真实数据，具有交互性，并且可以更加详细。我发现这一点非常吸引人，并尽可能使用 Framer。目前唯一的缺点是缺少网页版本。

**[Figma](https://www.figma.com/)**

Figma 是协作设计的标准。它使用起来很直观，拥有插件生态系统，并且性能无与伦比。当我需要与其他设计师或开发人员在同一个文件中协作时，我会使用它。

**[Adobe Creative Cloud](https://www.adobe.com/creativecloud.html)**

我和女朋友共享一个 Adobe Cloud 的月度订阅。我只使用 Lightroom、Photoshop 和 Illustrator。我认为大多数 Adobe 应用程序都很笨重，功能过于强大，我使用时经常感到沮丧。

**[Canva](https://www.canva.com/)**

Canva 是一个简单但功能丰富的网络应用程序，适用于快速设计。它拥有大量的模板库，涵盖你能想到的所有格式。这使得你可以快速设计出看起来很棒的作品。我使用 Canva 进行社交媒体帖子、印刷设计和其他随机设计。

## 开发

**[VS Code](https://code.visualstudio.com/)**

我开始编码时是在 2018 年底，当时 VS Code 已经非常流行。这绝非偶然。这个应用程序快如闪电，并且提供了许多社区扩展。我所有的编码工作都在 VS Code 中完成，没有任何抱怨。

**[Netlify](https://www.netlify.com/)**

在主机和后端主题方面，我不是专家。这就是为什么我喜欢 Netlify。它简单易用，让我能够回到自己熟悉的领域。我使用免费套餐，并用 Netlify 托管我的静态项目。我还有一双 Netlify 送的漂亮袜子。

**[Namecheap](https://www.namecheap.com/)**

我第一次购买域名是在 Namecheap，之后再也没有更换过。可能还有更好的域名注册商，但我一直没有勇气更换。如果你认为我需要更换，请告诉我。

**[GitHub](https://github.com/)**我的所有代码都托管在 GitHub 上。你可以在这里查看这个网站的代码：[这里](https://github.com/wirtzdan/new-website)。如果你不理解某些内容，随时可以给我写信。我会很乐意帮助你。

**[iTerm 2](https://iterm2.com/)**

我使用 iTerm 作为我的终端客户端，并与 zsh 配合使用。老实说，除了这里有漂亮的主题外，我也不知道为什么需要它。

**[ImageOptim](https://imageoptim.com/mac)**

ImageOptim 是一个隐藏的宝石。这个小应用可以轻松批量压缩 100 张图片，并且通常可以将文件大小减半。我将其与预览（Preview）结合使用，来裁剪、缩放和压缩所有用于网络使用的图片。

## 技术装备

**[Peak Design Everyday Backpack](https://www.peakdesign.com/products/everyday-backpack)**

Peak Design 是我愿意为之赴汤蹈火的品牌之一。一开始我犹豫是否要购买这个背包，但事实证明它物有所值。我每天都背着这个背包，用于工作、旅行、购物和摄影。

**[MacBook Pro（15 英寸，2017 年款）](https://www.apple.com/shop/buy-mac/macbook-pro)**

我的移动工作站和背包伴侣。我大部分时间都在使用这个设备，并享受它带来的便利。虽然价格不菲，但值得投资。

**[富士 X-T10 相机](https://www.fujifilmusa.com/products/digital_cameras/x/fujifilm_x_t10/)**

我与这款相机有着特殊的情感联系，因为我是在大学开始时购买的它。它体积小巧，按键手感出色，并且总能以出色的成像质量让我惊喜。我经常在两个定焦镜头（23mm F2 和 50mm F2）之间切换。

**[Magic Mouse 2](https://www.apple.com/de/shop/product/MLA02Z/A/magic-mouse-2-silber)**

我在 eBay 上以 30 欧元的价格淘到的 Apple Magic Mouse 2。它在任何表面上都能完美使用。唯一的不足是它需要电池。我经常在外出时忘记带备用电池。

**[人体工学笔记本电脑支架](https://www.amazon.de/dp/B01N99IGMQ/ref=sr_1_5?crid=2XJA82MXCVRRC&keywords=mobiler+laptop+st%C3%A4nder&qid=1573384409&sprefix=mobile+laptop%2Caps%2C200&sr=8-5)**

这是我找到的最小巧、最轻便的笔记本电脑支架。它总是固定在我的背包侧面，并且可以快速设置。

**[罗技 K380 键盘](https://www.logitech.com/en-us/product/multi-device-keyboard-k380)**

这个键盘[完美适合](https://twitter.com/wirtzdan/status/1193482720615243776?s=19)放在我的背包顶部。仅这一点就让我非常满意。结合我的笔记本电脑支架、鼠标和背包，我拥有一个非常灵活的工作站。

# Introduction

Le web décentralisé connaît une croissance rapide, mais l’un des plus grands défis pour les développeurs reste de trouver un moyen d’héberger leurs frontends de façon sécurisée et résistante à la censure. dAppling Network est une nouvelle plateforme qui vise à résoudre ce problème en offrant une solution simple et facile à utiliser pour héberger des frontends décentralisés.

# Qu'est-ce que dAppling Network ?

dAppling Network est un service d’hébergement web conçu spécifiquement pour les protocoles Web3. Il utilise l’InterPlanetary File System (IPFS) pour stocker et diffuser le code des frontends, ce qui le rend à la fois décentralisé et résistant à la censure. dAppling offre également plusieurs autres fonctionnalités qui en font un outil précieux pour les développeurs Web3, notamment :

- **Déploiements vérifiables :** dAppling utilise un processus de déploiement vérifiable garantissant que votre code frontend est correctement déployé et ne peut être altéré.
- **Frontends contrôlés par des smart contracts :** dAppling vous permet de gérer vos frontends via des smart contracts, offrant ainsi un contrôle accru sur vos applications.
- **Performances :** en utilisant IPFS pour diffuser votre code, dAppling assure des performances optimales.

# Comment fonctionne dAppling ?

Pour utiliser dAppling, il vous suffit de créer un compte et de télécharger votre code frontend. La plateforme déploiera ensuite votre code sur IPFS et générera une URL que vous pourrez utiliser pour accéder à votre frontend.

# Pourquoi utiliser dAppling Network ?

Voici quelques raisons d’utiliser dAppling Network pour héberger vos frontends :

- **Résistance à la censure :** grâce à l’utilisation d’IPFS, votre code frontend ne peut être censuré par des gouvernements ou autres organisations.
- **Sécurité :** dAppling intègre de nombreuses fonctionnalités de sécurité, telles que les déploiements vérifiables et le contrôle via smart contracts, pour protéger vos frontends.
- **Performances :** la diffusion du code via IPFS garantit une grande rapidité et fiabilité.
- **Facilité d'utilisation :** la plateforme est très simple d’utilisation : créez un compte et déployez votre frontend en quelques minutes.

# Introduction

Le web décentralisé se développe rapidement, mais l'un des plus grands défis pour les développeurs est de trouver un moyen d'héberger leurs interfaces frontales de manière résistante à la censure et sécurisée. **dAppling Network** est une nouvelle plateforme qui vise à résoudre ce problème en offrant une solution simple et facile à utiliser pour héberger des interfaces décentralisées.

# Qu'est-ce que dAppling Network ?

**dAppling Network** est un service d'hébergement web spécialement conçu pour les protocoles Web3. Il utilise le **InterPlanetary File System (IPFS)** pour stocker et servir le code frontal, ce qui le rend résistant à la censure et décentralisé. dAppling propose également plusieurs autres fonctionnalités qui en font un outil précieux pour les développeurs Web3, notamment :

- **Déploiements vérifiables** : dAppling utilise un processus de déploiement vérifiable qui garantit que votre code frontal est déployé correctement et qu'il ne peut pas être altéré.
- **Frontaux contrôlés par des contrats intelligents** : dAppling vous permet de contrôler vos interfaces frontales à l'aide de contrats intelligents, ce qui vous donne encore plus de contrôle sur vos applications.
- **Performance** : dAppling utilise IPFS pour servir votre code frontal, ce qui le rend très performant.

# Comment fonctionne dAppling ?

Pour utiliser dAppling, il vous suffit de créer un compte et de télécharger votre code frontal. dAppling déployera ensuite votre code sur IPFS et générera une URL que vous pourrez utiliser pour accéder à votre interface.

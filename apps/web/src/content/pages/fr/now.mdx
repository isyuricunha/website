### Projets

<NowProject repo='isyuricunha/yumusic' name='YuMusic'>
  Un client musical Subsonic open-source et léger, conçu nativement pour Android. Axé sur une
  expérience d'écoute épurée, avec des fonctionnalités telles que la navigation/recherche dans la
  bibliothèque, la gestion des listes de lecture, la lecture sans interruption, Android Auto et des
  intégrations optionnelles (par exemple, le scrobbling). Le mode hors ligne est en cours de
  développement actif.
</NowProject>

<NowProject repo='isyuricunha/yuflow' name='Yuflow' website='https://yuflow.yuricunha.com/'>
  Un gestionnaire de tâches axé sur la confidentialité, où vos données restent sur votre appareil.
  Conçu pour le bureau (Tauri) et le web (PWA), avec une recherche locale rapide, des tags/filtres,
  un comportement hors ligne prioritaire et une interface utilisateur minimaliste et accessible.
</NowProject>

<NowProject repo='isyuricunha/YuPomo' name='YuPomo' website='https://yupomo.yuricunha.com/'>
  Un minuteur Pomodoro élégant et minimaliste, disponible en tant qu'application web et application
  native pour bureau (Tauri). Cycles personnalisables, persistance locale, statistiques de
  productivité, notifications et support multilingue.
</NowProject>

<NowProject
  repo='isyuricunha/sql-query-analyzer'
  name='SQL Query Analyzer'
  website='<https://sql-query-analyzer.yuricunha.com/>'
>
  Un outil basé sur le web qui analyse les requêtes SQL, met en évidence les pièges de performance
  courants et suggère des optimisations (ainsi que les meilleures pratiques générales) pour les
  administrateurs de bases de données, les développeurs et les analystes soucieux d'écrire du SQL
  efficace.
</NowProject>

<NowProject
  repo='isyuricunha/yue-discord-bot'
  name='YueBot'
  website='https://yuebot.yuricunha.com/'
>
  YueBot est conçu pour être opéré avec un panneau d'administration web plutôt que de dépendre
  uniquement des commandes de chat. En pratique, le bot et le panneau partagent une seule base de
  données PostgreSQL et un client Prisma commun.
</NowProject>

<NowProject repo='isyuricunha/website' name='website' website='https://yuricunha.com/'>

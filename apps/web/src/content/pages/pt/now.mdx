### Projetos

<NowProject repo='isyuricunha/yumusic' name='YuMusic'>
  Um cliente Subsonic open source e leve, feito nativamente para Android. A ideia é ter uma
  experiência de ouvir música bem limpa, com navegação/busca na biblioteca, playlists, gapless
  playback, Android Auto e integrações opcionais (ex.: scrobbling). O modo offline está em
  desenvolvimento ativo.
</NowProject>

<NowProject repo='isyuricunha/yuflow' name='Yuflow' website='https://yuflow.yuricunha.com/'>
  Um gerenciador de tarefas privacy-first onde seus dados ficam no seu dispositivo. Feito para
  desktop (Tauri) e web (PWA), com busca local rápida, tags/filtros, funcionamento offline-first e
  uma UI minimalista e acessível.
</NowProject>

<NowProject repo='isyuricunha/YuPomo' name='YuPomo' website='https://yupomo.yuricunha.com/'>
  Um timer Pomodoro bonito e minimalista, disponível como web app e como app desktop nativo (Tauri).
  Tem ciclos configuráveis, persistência local, estatísticas de produtividade, notificações e
  suporte a múltiplos idiomas.
</NowProject>

<NowProject
  repo='isyuricunha/sql-query-analyzer'
  name='SQL Query Analyzer'
  website='<https://sql-query-analyzer.yuricunha.com/>'
>
  Uma ferramenta web que analisa queries SQL, aponta problemas comuns de performance e sugere
  otimizações (e boas práticas) para DBAs, devs e analistas que se importam em escrever SQL
  eficiente.
</NowProject>

<NowProject
  repo='isyuricunha/yue-discord-bot'
  name='YueBot'
  website='https://yuebot.yuricunha.com/'
>
  O YueBot foi projetado para ser operado com um painel de administração web em vez de depender
  apenas de comandos de chat. Na prática, o bot e o painel compartilham um único banco de dados
  PostgreSQL e um cliente Prisma compartilhado.
</NowProject>

<NowProject repo='isyuricunha/website' name='website' website='https://yuricunha.com/'>
  O código-fonte do meu site pessoal (yuricunha.com), organizado como monorepo. Inclui o app web em
  Next.js, packages compartilhados (UI, i18n, emails, utils) e backend com tRPC e Drizzle/Postgres.
</NowProject>

## O que eu estou fazendo agora

Esta página é um retrato do que eu estou focando no momento.

### Trabalho

- Mantendo e melhorando infraestrutura de servidores e bancos de dados
- Trabalhando em confiabilidade, performance e observabilidade

### Projetos

- **[YuMusic](https://github.com/isyuricunha/yumusic)** — Um cliente Subsonic open source e leve, feito nativamente para Android. A ideia é ter uma experiência de ouvir música bem limpa, com navegação/busca na biblioteca, playlists, gapless playback, Android Auto e integrações opcionais (ex.: scrobbling). O modo offline está em desenvolvimento ativo.
- **[Yuflow](https://github.com/isyuricunha/yuflow)** — Um gerenciador de tarefas privacy-first onde seus dados ficam no seu dispositivo. Feito para desktop (Tauri) e web (PWA), com busca local rápida, tags/filtros, funcionamento offline-first e uma UI minimalista e acessível.
- **[YuPomo](https://github.com/isyuricunha/YuPomo)** — Um timer Pomodoro bonito e minimalista, disponível como web app e como app desktop nativo (Tauri). Tem ciclos configuráveis, persistência local, estatísticas de produtividade, notificações e suporte a múltiplos idiomas.
- **[SQL Query Analyzer](https://github.com/isyuricunha/sql-query-analyzer)** — Uma ferramenta web que analisa queries SQL, aponta problemas comuns de performance e sugere otimizações (e boas práticas) para DBAs, devs e analistas que se importam em escrever SQL eficiente.
- **[website](https://github.com/isyuricunha/website)** — O código-fonte do meu site pessoal (yuricunha.com), organizado como monorepo. Inclui o app web em Next.js, packages compartilhados (UI, i18n, emails, utils) e backend com tRPC e Drizzle/Postgres.

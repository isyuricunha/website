## En qué estoy trabajando ahora

Esta página es un resumen de en qué me estoy enfocando ahora mismo.

- Trabajando en confiabilidad, rendimiento y observabilidad

### Proyectos

<NowProject repo='isyuricunha/yumusic' name='YuMusic'>
  Un cliente de música Subsonic de código abierto y liviano, construido de forma nativa para
  Android. Enfocado en una experiencia de escucha limpia, con funciones como navegación/búsqueda en
  la biblioteca, gestión de listas de reproducción, reproducción continua, Android Auto e
  integraciones opcionales (por ejemplo, scrobbling). El modo sin conexión está en desarrollo
  activo.
</NowProject>

<NowProject repo='isyuricunha/yuflow' name='Yuflow' website='https://yuflow.yuricunha.com/'>
  Un gestor de tareas con privacidad primero, donde tus datos permanecen en tu dispositivo.
  Construido para escritorio (Tauri) y la web (PWA), con búsqueda local rápida, etiquetas/filtros,
  comportamiento offline-first y una interfaz de usuario accesible y minimalista.
</NowProject>

<NowProject repo='isyuricunha/YuPomo' name='YuPomo' website='https://yupomo.yuricunha.com/'>
  Un temporizador Pomodoro hermoso y minimalista, disponible como aplicación web y aplicación de
  escritorio nativa (Tauri). Ciclos personalizables, persistencia local, estadísticas de
  productividad, notificaciones y soporte multilingüe.
</NowProject>

<NowProject
  repo='isyuricunha/sql-query-analyzer'
  name='Analizador de Consultas SQL'
  website='<https://sql-query-analyzer.yuricunha.com/>'
>
  Una herramienta basada en web que analiza consultas SQL, destaca problemas comunes de rendimiento
  y sugiere optimizaciones (y buenas prácticas generales) para administradores de bases de datos,
  desarrolladores y analistas que se preocupan por escribir SQL eficiente.
</NowProject>

<NowProject
  repo='isyuricunha/yue-discord-bot'
  name='YueBot'
  website='https://yuebot.yuricunha.com/'
>
  YueBot está diseñado para operarse con un panel de administración web en lugar de depender solo de
  comandos de chat. En la práctica, el bot y el panel comparten una única base de datos PostgreSQL y
  un cliente Prisma compartido.
</NowProject>

<NowProject repo='isyuricunha/website' name='sitio web' website='https://yuricunha.com/'>
  El código fuente de mi sitio web personal (yuricunha.com), construido como un monorepositorio.
  Incluye la aplicación web Next.js, paquetes compartidos (UI, i18n, emails, utils) y componentes de
  backend como tRPC y Drizzle/Postgres.
</NowProject>

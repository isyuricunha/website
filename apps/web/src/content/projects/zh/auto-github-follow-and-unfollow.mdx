## 项目简介

**Auto GitHub Follow and Unfollow** 是一款用于简化 GitHub 关系管理的轻量级用户脚本。无论你想拓展人脉，还是清理正在关注的列表，它都能在个人资料页以及关注者/正在关注页面上自动执行关注与取消关注。

### 功能特性

- **自动关注**：无需逐个打开页面，即可快速关注多个账号。
- **高效取关**：支持连续取消关注，节省管理时间。
- **易于使用**：无缝融入 GitHub 个人资料页与关注者/正在关注列表。

### 工作原理

脚本基于 JavaScript 开发，通过 Tampermonkey（常用的用户脚本管理扩展）运行。安装后，它会在 GitHub 界面中增加相应功能，让关注/取消关注操作更省力。兼容 Chromium 系浏览器（如 Chrome、Edge、Brave、Opera）以及 Firefox。

### 为什么选择它

- **高效**：适合需要管理大量关系或经常与众多账号互动的用户。
- **简单**：无需复杂配置——在 Tampermonkey 中安装即可使用。
- **保持更新**：至 2025 年仍可用，已在最新 GitHub 界面中测试。

### 入门指南

使用前你需要：
1. 在浏览器中安装 Tampermonkey 扩展；
2. 从[项目 GitHub 页面](https://github.com/isyuricunha/auto-github-follow-and-unfollow)获取脚本；
3. 按仓库 README 的说明完成安装与设置。

## Sobre el Proyecto

**Auto GitHub Follow and Unfollow** es un *userscript* ligero diseñado para simplificar la gestión de tus conexiones en GitHub. Ya sea que desees expandir tu red o limpiar tu lista de seguidos, este script automatiza el proceso de seguir y dejar de seguir usuarios directamente desde los perfiles de GitHub o las páginas de seguidores/seguidos.

### ¿Qué Hace?

- **Automatiza el Seguimiento**: Sigue rápidamente a múltiples usuarios sin necesidad de hacer clic manualmente en cada perfil.
- **Simplifica el Dejar de Seguir**: Deja de seguir usuarios de manera masiva, ahorrando tiempo al gestionar tus conexiones.
- **Amigable para el Usuario**: Funciona sin problemas en las páginas de perfil de GitHub y en las listas de seguidores/seguidos, integrándose en tu experiencia de navegación.

### ¿Cómo Funciona?

Este script está desarrollado en JavaScript y se ejecuta a través de **Tampermonkey**, una extensión popular para navegadores que permite el uso de *userscripts*. Una vez instalado, añade funcionalidades a la interfaz de GitHub, permitiéndote realizar acciones de seguir y dejar de seguir con un esfuerzo mínimo. Es compatible con navegadores basados en Chromium (como Chrome, Edge, Brave y Opera) y Firefox, garantizando una amplia accesibilidad.

### ¿Por Qué Usarlo?

- **Eficiencia**: Ideal para usuarios que gestionan redes grandes o interactúan con muchos perfiles de GitHub de manera regular.
- **Simplicidad**: Sin configuraciones complejas, solo instala el script a través de Tampermonkey y comienza a usarlo.
- **Actualizado**: Totalmente funcional hasta 2025, probado para trabajar con la última interfaz de GitHub.

### Cómo Empezar

Para usar el script, necesitarás:

1. La extensión Tampermonkey instalada en tu navegador.
2. El script en sí, disponible en la [página del proyecto en GitHub](https://github.com/isyuricunha/auto-github-follow-and-unfollow).
3. Sigue las instrucciones de instalación en el archivo README del repositorio para configurarlo.

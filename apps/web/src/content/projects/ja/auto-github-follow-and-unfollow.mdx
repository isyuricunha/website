## プロジェクトについて

**Auto GitHub Follow and Unfollow** は、GitHub の接続管理を簡素化するために設計された軽量なユーザースクリプトです。ネットワークを拡大したい場合でも、フォローリストを整理したい場合でも、このスクリプトは GitHub のプロファイルページやフォロワー/フォロー中ページから直接ユーザーのフォローとフォロー解除を自動化します。

### 機能

- **フォローの自動化**: 各プロファイルを手動でクリックしなくても、複数のユーザーを簡単にフォローできます。
- **フォロー解除の効率化**: 一括でユーザーのフォローを解除し、接続管理の時間を節約します。
- **ユーザーフレンドリー**: GitHub のプロファイルページやフォロワー/フォロー中リストでシームレスに動作し、ブラウジング体験に統合されます。

### 動作原理

このスクリプトは JavaScript で動作し、ユーザースクリプト用の人気ブラウザ拡張機能である Tampermonkey を介して実行されます。インストール後、GitHub のインターフェースに機能が追加され、最小限の労力でフォローとフォロー解除の操作が可能になります。Chromium ベースのブラウザ（Chrome、Edge、Brave、Opera など）および Firefox と互換性があり、幅広いアクセシビリティを確保します。

### 利用する理由

- **効率性**: 大規模なネットワークを管理したり、定期的に多くの GitHub プロファイルとやり取りするユーザーに最適です。
- **簡単さ**: 複雑な設定は不要で、Tampermonkey を介してスクリプトをインストールするだけで使用できます。
- **最新性**: 2025 年時点でも完全に機能し、GitHub の最新インターフェースで動作することがテスト済みです。

### 使い始める

スクリプトを使用するには、以下が必要です：

1. ブラウザに Tampermonkey 拡張機能がインストールされていること。
2. スクリプト自体（[プロジェクトの GitHub ページ](https://github.com/isyuricunha/auto-github-follow-and-unfollow)で入手可能）。
3. リポジトリの README に記載されているインストール手順に従って設定します。

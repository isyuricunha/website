## プロジェクトについて

**Auto GitHub Follow and Unfollow** は、GitHub 上のつながり管理をシンプルにするための軽量ユーザースクリプトです。ネットワークを広げたいときも、フォロー中の整理をしたいときも、プロフィールやフォロワー／フォロー中ページからのフォロー／アンフォロー操作を自動化します。

### できること

- **フォローの自動化**: 複数ユーザーを素早くフォロー。
- **アンフォローの効率化**: 一括アンフォローで管理時間を短縮。
- **使いやすさ**: プロフィールやフォロワー／フォロー中一覧に自然に組み込まれて動作。

### 使い方

このスクリプトは JavaScript 製で、ユーザースクリプトマネージャの Tampermonkey 上で動作します。インストールすると GitHub の UI に機能が追加され、最小限の操作でフォロー／アンフォローが可能になります。Chromium 系ブラウザ（Chrome、Edge、Brave、Opera など）および Firefox に対応しています。

### なぜ使うのか

- **効率的**: 多くのプロフィールに定期的に関わるユーザーに最適。
- **シンプル**: 複雑な設定は不要。Tampermonkey に読み込むだけで開始できます。
- **最新**: 2025 年時点でも最新の GitHub インターフェースで動作確認済み。

### はじめ方

必要なもの:
1. ブラウザに Tampermonkey 拡張をインストール
2. [プロジェクトの GitHub ページ](https://github.com/isyuricunha/auto-github-follow-and-unfollow) からスクリプトを取得
3. リポジトリの README に従ってセットアップ

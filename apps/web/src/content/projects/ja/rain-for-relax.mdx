**Rain for Relax** は、静かに集中したいときや穏やかに過ごしたいときに最適な、落ち着く体験を提供するミニマルな Web アプリです。厳選したチルミュージックと優しい雨音で、シンプルかつ途切れない没入感をお届けします。

再生ボタンを押すだけ。目を閉じて、音の空間に身を委ねてください。

### 🌟 主な特徴

- 心地よいチルトラックのセレクション
- 余計な要素のないクリーンな UI
- 途切れない連続再生
- 没入感を生む高音質オーディオ
- モダンなブラウザで誰でも使えるオープンソース

### 🚀 使い方

セットアップは不要です — そのまま [サイトへアクセス](https://www.rain-for-relax.yuricunha.com) して再生を押すだけ。ローカルで動かす場合は、リポジトリをクローンして `index.html` を開けば OK です。

### 🤝 コントリビュート

改善案や音源の追加提案があれば、フォークして変更を加え、プルリクエストを送ってください。気軽にどうぞ。

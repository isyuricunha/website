## プロジェクトについて

YuPomo は、意図的に働き、意図的に休むためのフォーカス重視のポモドーロタイマーです。高速な Web アプリとして、また Tauri により Windows / macOS / Linux 向けのネイティブデスクトップアプリとして動作します。インターフェースはシンプルで、余計な手間なくすぐにセッションを開始できます。設定や統計はローカルに保存され、あなたのデータはデバイス内に留まります。

YuPomo はクラシックなポモドーロ手法に従い、初期値は「作業 25 分・短い休憩 5 分・長い休憩 15 分」です。各時間は自由に変更でき、サウンドアラートや通知、ライト／ダークテーマの切り替えにも対応。統計ビューでは日・週・月ごとの集中履歴をチャートで可視化でき、傾向を把握して習慣改善に役立てられます。

## 主な機能

作業／短い休憩／長い休憩の一連のポモドーロフロー。純黒のダークテーマを含むテーマ切り替え。時間・サウンド・通知の細かな設定。設定と履歴のローカル永続化。集中時間とセッション数のチャート表示による統計。デスクトップ／モバイルに最適化されたレスポンシブ UI。ランタイムで即時反映される多言語化。デスクトップでは新リリースを知らせる任意のアップデートバナーに対応。

## 対応プラットフォームとインストール

Web 版はデモリンクから利用できます。デスクトップアプリは GitHub Releases で配布しており、Windows / macOS / Linux 用のインストーラを用意しています。Windows では setup exe または msi、macOS ではチップに合わせた dmg、Linux では deb / rpm に加え AppImage も利用可能です。デスクトップでは更新チェックにも対応し、最新バージョンのダウンロードとインストールを案内します。

## 開発とビルド

フロントエンドは React + TypeScript を Vite で開発し、デスクトップシェルに Tauri を用いています。リポジトリをクローンし、パッケージマネージャで依存関係をインストールすれば、Web の開発サーバーや Tauri のデスクトップ開発プロセスを起動できます。Web デプロイと各 OS 向けデスクトップバンドルのビルドスクリプトも同梱。バージョン同期スクリプトにより Web と Tauri のバージョン整合を保ち、リリースワークフローはアーティファクトをビルドして GitHub Releases へ公開します。

## デスクトップ自動アップデート

デスクトップ版は Tauri Updater に対応。最新の GitHub リリースを確認し、アプリ内バナーで新バージョンを通知、ダウンロードとインストールを行えます。アップデータの成果物は署名・検証済み。Tauri 設定の公開鍵とリリースの latest.json を用いた構成で、設定から自動インストールの有効化も可能です。

## プライバシー

YuPomo は個人データを収集しません。設定と統計はローカル保存です。Web 版は localStorage、デスクトップ版は Tauri のアプリデータディレクトリ配下のファイルシステムを利用します。通知とサウンドはローカル生成。任意の更新チェックは公開 GitHub Releases API へのアクセスのみです。

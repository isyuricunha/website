## プロジェクトについて

YuPomo は、集中して作業し、意図的に休憩するためのポモドーロタイマーです。高速なウェブアプリとして、また Tauri を通じて Windows、macOS、Linux 用のネイティブデスクトップアプリとして動作します。インターフェースはシンプルで直感的なので、セッションをスムーズに開始し、集中力を維持できます。設定と統計データはローカルに保存されるため、データはデバイス内に留まります。

YuPomo は、クラシックなポモドーロ手法に従い、作業時間はデフォルトで25分、短い休憩時間は5分、長い休憩時間は15分に設定されています。すべての時間を変更したり、サウンドアラートや通知を有効にしたり、ライトテーマとダークテーマを切り替えたりすることができます。統計ビューでは、日次、週次、月次の集中履歴をチャートで表示し、傾向を観察してルーチンを改善することができます。

## 主な機能

- 作業、短い休憩、長い休憩を含む完全なポモドーロフロー
- 真のブラックダークテーマを含むテーマ切り替え
- 時間、サウンド、通知の設定可能
- 設定と履歴のローカル保存
- 集中時間とセッション数を視覚化するチャート付き統計
- デスクトップとモバイルに対応したレスポンシブレイアウト
- 実行時に即座に言語を切り替えられる多言語インターフェース
- 新しいリリースが利用可能な場合に通知するオプションのデスクトップ更新バナー

## プラットフォームとインストール

ウェブ版はデモリンクから利用可能です。デスクトップアプリは、GitHub Releases を通じて Windows、macOS、Linux 用のインストーラーで配布されています。Windows では setup exe または msi インストーラーを使用できます。macOS では、チップに応じた dmg を使用してインストールします。Linux では deb または rpm パッケージを選択するか、AppImage を実行できます。また、デスクトップアプリは更新をチェックし、最新バージョンのダウンロードとインストールを案内します。

## 開発とビルド

このプロジェクトは、ウェブクライアントに React と TypeScript、Vite を使用し、デスクトップシェルに Tauri を使用しています。リポジトリをクローンし、パッケージマネージャーで依存関係をインストールして、ウェブ用の開発サーバーまたはデスクトップ用の Tauri 開発プロセスを実行できます。ビルドスクリプトは、ウェブデプロイメントと各オペレーティングシステム用のデスクトップバンドル用に提供されています。バージョニングスクリプトは、ウェブと Tauri のバージョンを同期し、リリースワークフローはアーティファクトをビルドして GitHub Releases に公開します。

## デスクトップの自動更新

デスクトップビルドは Tauri Updater をサポートしています。アプリは最新の GitHub リリースをチェックし、新しいバージョンのインアプリバナーを表示してダウンロードとインストールを行います。アップデーターのアーティファクトは署名され、検証されます。設定は Tauri の設定内の公開キーとリリース上の最新の JSON ファイルを使用します。デスクトップでは、設定で起動時に自動インストールを有効にできます。

## プライバシー

YuPomo は個人データを収集しません。設定と統計はローカルに保存されます。ウェブ版は localStorage を使用し、デスクトップ版は Tauri ファイルシステムを使用してアプリデータディレクトリに保存します。通知とサウンドはローカルで生成されます。オプションの更新チェックは、公開 GitHub Releases API にのみアクセスします。

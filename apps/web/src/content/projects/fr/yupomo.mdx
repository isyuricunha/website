## À propos du projet

YuPomo est un minuteur Pomodoro conçu pour faciliter des cycles de travail efficaces et des pauses intentionnelles. Il fonctionne comme application Web rapide et comme application de bureau native pour Windows, macOS et Linux grâce à Tauri. L interface est claire et directe afin de lancer une session sans friction et de rester concentré. Les réglages et les statistiques sont enregistrés localement, vos données restent sur votre appareil.

YuPomo suit la méthode Pomodoro classique avec des valeurs par défaut de vingt cinq minutes de travail, cinq minutes de courte pause et quinze minutes de longue pause. Vous pouvez modifier toutes les durées, activer les alertes sonores et les notifications, et choisir un thème clair ou sombre. Une vue statistiques présente votre historique quotidien, hebdomadaire et mensuel avec des graphiques pour observer les tendances et améliorer votre routine.

## Principales fonctionnalités

Cycle Pomodoro complet avec travail, courte pause et longue pause. Changement de thème avec noir total en mode sombre. Durées, sons et notifications configurables. Persistance locale pour les préférences et l historique. Statistiques avec graphiques pour visualiser le temps de concentration et le nombre de sessions. Interface réactive pour bureau et mobile. Interface multilingue avec changement de langue instantané à l exécution. Bandeau de mise à jour optionnel sur le bureau pour avertir des nouvelles versions.

## Plates formes et installation

La version Web est disponible via le lien de démonstration. L application de bureau est distribuée via GitHub Releases avec des installateurs pour Windows, macOS et Linux. Sous Windows vous pouvez utiliser l installateur exe ou msi. Sous macOS installez via l image dmg adaptée à votre puce. Sous Linux choisissez les paquets deb ou rpm ou utilisez le fichier AppImage. L application peut vérifier les mises à jour sur le bureau et vous guider pour installer la dernière version.

## Développement et build

Le projet utilise React et TypeScript avec Vite pour le client Web et Tauri pour l enveloppe de bureau. Vous pouvez cloner le dépôt, installer les dépendances et lancer le serveur de développement pour le Web ou le processus Tauri pour le bureau. Des scripts de build sont fournis pour le déploiement Web et pour les paquets bureau pour chaque système. Les scripts de version maintiennent la synchronisation entre le Web et Tauri et le flux de publication construit et publie les artefacts sur GitHub Releases.

## Mise à jour automatique sur le bureau

Les builds bureau prennent en charge Tauri Updater. L application peut vérifier la dernière version GitHub, afficher un bandeau dans l interface, puis télécharger et installer la nouvelle version. Les artefacts de mise à jour sont signés et vérifiés. La configuration utilise une clé publique dans la configuration Tauri et le fichier latest json de la publication. Vous pouvez activer l installation automatique au démarrage dans les paramètres sur le bureau.

## Confidentialité

YuPomo ne collecte pas de données personnelles. Les préférences et les statistiques sont enregistrées localement. Sur le Web il s agit de localStorage. Sur le bureau l application utilise le système de fichiers Tauri dans le répertoire des données de l application. Les notifications et les sons sont générés localement. La vérification de mise à jour optionnelle n interroge que l API publique GitHub Releases.

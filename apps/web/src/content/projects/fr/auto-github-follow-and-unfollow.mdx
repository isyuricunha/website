## À propos du projet

**Auto GitHub Follow and Unfollow** est un userscript léger conçu pour simplifier la gestion de vos connexions GitHub. Que vous souhaitiez développer votre réseau ou nettoyer votre liste de suivis, ce script automatise le processus de suivi et de cessation de suivi des utilisateurs directement depuis les profils GitHub ou les pages des abonnés/abonnements.

### Fonctionnalités

- **Automatise les suivis** : Suivez rapidement plusieurs utilisateurs sans avoir à cliquer manuellement sur chaque profil.
- **Simplifie les cessations de suivi** : Arrêtez de suivre plusieurs utilisateurs en une seule fois, ce qui vous fait gagner du temps dans la gestion de vos connexions.
- **Convivial** : Fonctionne de manière transparente sur les pages de profil GitHub et les listes d'abonnés/abonnements, s'intégrant parfaitement à votre expérience de navigation.

### Fonctionnement

Ce script est alimenté par JavaScript et s'exécute via Tampermonkey, une extension de navigateur populaire pour les userscripts. Une fois installé, il ajoute des fonctionnalités à l'interface de GitHub, vous permettant d'effectuer des actions de suivi et de cessation de suivi avec un minimum d'effort. Il est compatible avec les navigateurs basés sur Chromium (comme Chrome, Edge, Brave et Opera) ainsi qu'avec Firefox, garantissant une accessibilité étendue.

### Pourquoi l'utiliser ?

- **Efficacité** : Idéal pour les utilisateurs qui gèrent de grands réseaux ou interagissent régulièrement avec de nombreux profils GitHub.
- **Simplicité** : Aucune configuration complexe — il suffit d'installer le script via Tampermonkey et de commencer à l'utiliser.
- **À jour** : Toujours pleinement fonctionnel en 2025, testé pour fonctionner avec la dernière interface de GitHub.

### Pour commencer

Pour utiliser le script, vous aurez besoin de :

1. L'extension Tampermonkey installée dans votre navigateur.
2. Le script lui-même, disponible sur la [page GitHub du projet](https://github.com/isyuricunha/auto-github-follow-and-unfollow).
3. Suivez les instructions d'installation dans le fichier README du dépôt pour le configurer.

## À propos du projet

**Auto GitHub Suivre et Ne plus suivre** est un userscript léger conçu pour simplifier la gestion de vos relations sur GitHub. Que vous souhaitiez développer votre réseau ou nettoyer votre liste d’abonnements, ce script automatise le suivi et le fait de ne plus suivre directement depuis les profils GitHub et les pages Abonnés/Abonnements.

### Ce que ça fait

- **Automatise le suivi** : suivez rapidement plusieurs comptes sans ouvrir chaque profil.
- **Facilite le désabonnement** : ne plus suivre en série pour gagner du temps dans la gestion de vos connexions.
- **Convivial** : s’intègre naturellement aux pages de profil et aux listes Abonnés/Abonnements de GitHub.

### Fonctionnement

Ce script est écrit en JavaScript et fonctionne via Tampermonkey, une extension de navigateur populaire pour les userscripts. Une fois installé, il ajoute des fonctionnalités à l’interface de GitHub pour suivre/ne plus suivre avec un minimum d’effort. Il est compatible avec les navigateurs basés sur Chromium (Chrome, Edge, Brave, Opera) ainsi qu’avec Firefox.

### Pourquoi l’utiliser ?

- **Efficacité** : idéal pour celles et ceux qui gèrent un large réseau ou interagissent régulièrement avec de nombreux profils.
- **Simplicité** : aucune configuration complexe — installez le script via Tampermonkey et commencez à l’utiliser.
- **À jour** : pleinement fonctionnel en 2025, testé avec la dernière interface de GitHub.

### Pour commencer

Pour utiliser le script, vous aurez besoin de :
1. L’extension Tampermonkey installée dans votre navigateur.
2. Le script lui‑même, disponible sur la [page GitHub du projet](https://github.com/isyuricunha/auto-github-follow-and-unfollow).
3. Suivez les instructions d’installation dans le README du dépôt.

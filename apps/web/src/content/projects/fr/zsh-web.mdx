J'ai beaucoup d'idées folles, et l'une d'elles est d'exécuter Zsh dans le navigateur. J'utilise Zsh depuis longtemps et je l'adore. Alors, j'essaie de créer Zsh Web. C'est un terminal basé sur le web qui exécute Zsh directement dans le navigateur. Il est développé avec Next.js et TypeScript.

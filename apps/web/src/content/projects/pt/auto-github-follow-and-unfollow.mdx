## Sobre o projeto

O **Auto GitHub Follow and Unfollow** é um userscript leve criado para simplificar a gestão das suas conexões no GitHub. Quer você queira expandir sua rede ou organizar a sua lista de seguidos, o script automatiza o processo de seguir e deixar de seguir diretamente em perfis e nas páginas de seguidores/seguidos.

### O que ele faz

- **Automatiza seguir**: Siga várias contas rapidamente sem abrir cada perfil manualmente.
- **Facilita deixar de seguir**: Deixe de seguir em sequência para economizar tempo na organização das conexões.
- **Amigável**: Funciona de forma integrada nas páginas de perfil e nas listas de seguidores/seguidos do GitHub.

### Como funciona

O script é escrito em JavaScript e roda via Tampermonkey, uma extensão popular para userscripts. Depois de instalado, ele adiciona recursos à interface do GitHub para que você siga/deixe de seguir com o mínimo de esforço. É compatível com navegadores baseados em Chromium (Chrome, Edge, Brave, Opera) e também com o Firefox.

### Por que usar?

- **Eficiência**: Perfeito para quem gerencia muitas conexões ou interage com vários perfis com frequência.
- **Simplicidade**: Sem configuração complexa — instale pelo Tampermonkey e comece a usar.
- **Atual**: Totalmente funcional em 2025 e testado na interface mais recente do GitHub.

### Primeiros passos

Para usar o script, você vai precisar de:
1. A extensão Tampermonkey instalada no seu navegador.
2. O script, disponível na [página do projeto no GitHub](https://github.com/isyuricunha/auto-github-follow-and-unfollow).

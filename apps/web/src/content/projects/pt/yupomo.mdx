## Sobre o projeto

O YuPomo é um temporizador Pomodoro focado que ajuda você a trabalhar por ciclos e descansar com intenção. Ele roda como aplicativo web rápido e como aplicativo desktop nativo para Windows, macOS e Linux por meio do Tauri. A interface é limpa e direta para iniciar uma sessão sem atrito e manter a concentração. As configurações e estatísticas são salvas localmente para que seus dados fiquem no seu dispositivo.

O YuPomo segue o método Pomodoro clássico com valores padrão de vinte e cinco minutos de trabalho, cinco minutos de pausa curta e quinze minutos de pausa longa. Você pode alterar todas as durações, ativar alertas sonoros e notificações e escolher entre tema claro e escuro. Uma visão de estatísticas mostra seu histórico diário, semanal e mensal com gráficos para observar tendências e melhorar sua rotina ao longo do tempo.

## Principais recursos

Fluxo Pomodoro completo com trabalho, pausa curta e pausa longa. Troca de tema com preto total no tema escuro. Durações, sons e notificações configuráveis. Persistência local para preferências e histórico. Estatísticas com gráficos para visualizar tempo de foco e contagem de sessões. Layout responsivo para desktop e celular. Interface multilíngue com troca instantânea de idioma em tempo de execução. Banner de atualização opcional no desktop que informa quando há nova versão.

## Plataformas e instalação

A versão web está disponível no link de demonstração. O aplicativo desktop é distribuído pelo GitHub Releases com instaladores para Windows, macOS e Linux. No Windows use o instalador exe ou msi. No macOS instale pelo arquivo dmg correspondente ao seu chip. No Linux escolha pacotes deb ou rpm ou utilize o AppImage. O aplicativo também pode verificar atualizações no desktop e orientar o download e a instalação da versão mais recente.

## Desenvolvimento e build

O projeto usa React e TypeScript com Vite para o cliente web e Tauri para o shell desktop. Você pode clonar o repositório, instalar dependências e executar o servidor de desenvolvimento para web ou o processo Tauri para desktop. Há scripts de build para deploy web e para empacotes de desktop por sistema operacional. Scripts de versionamento mantêm as versões de web e Tauri sincronizadas e o fluxo de release compila e publica os artefatos no GitHub Releases.

## Atualização automática no desktop

As compilações desktop suportam o Tauri Updater. O aplicativo pode verificar a versão mais recente no GitHub, exibir um banner na interface e baixar e instalar a nova versão. Os artefatos do updater são assinados e verificados. A configuração usa uma chave pública no arquivo de configuração do Tauri e o arquivo latest json da release. Você pode ativar a instalação automática na inicialização nas Configurações no desktop.

## Privacidade

O YuPomo não coleta dados pessoais. Preferências e estatísticas são salvas localmente. Na web usa se localStorage. No desktop o aplicativo usa o sistema de arquivos do Tauri no diretório de dados do aplicativo. Notificações e sons são gerados localmente. A verificação opcional de atualização usa apenas a API pública do GitHub Releases.

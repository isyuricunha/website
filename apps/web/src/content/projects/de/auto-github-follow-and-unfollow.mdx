## Über das Projekt

**Auto GitHub Follow and Unfollow** ist ein leichtgewichtiges Userscript, das das Verwalten Ihrer GitHub‑Verbindungen vereinfacht. Ob Sie Ihr Netzwerk ausbauen oder Ihre Following‑Liste aufräumen möchten – das Skript automatisiert das Folgen und Entfolgen direkt auf Profilseiten sowie den Follower-/Following‑Übersichten.

### Was es macht

- **Folgen automatisieren**: Schnell mehreren Accounts folgen, ohne jedes Profil einzeln aufrufen zu müssen.
- **Entfolgen vereinfachen**: Komfortables Entfolgen in Serie – spart Zeit beim Aufräumen der Verbindungen.
- **Benutzerfreundlich**: Funktioniert nahtlos auf Profilseiten sowie in den Follower-/Following‑Listen und integriert sich natürlich in den GitHub‑Ablauf.

### So funktioniert es

Das Skript ist in JavaScript geschrieben und läuft über Tampermonkey, eine verbreitete Browser‑Erweiterung für Userscripts. Nach der Installation erweitert es die GitHub‑Oberfläche, sodass Sie Follow‑ und Unfollow‑Aktionen mit minimalem Aufwand ausführen können. Es ist kompatibel mit Chromium‑basierten Browsern (z. B. Chrome, Edge, Brave, Opera) sowie Firefox.

### Warum dieses Tool?

- **Effizienz**: Ideal, wenn Sie viele Profile verwalten oder regelmäßig mit neuen Accounts interagieren.
- **Einfachheit**: Keine komplexe Einrichtung – Script via Tampermonkey installieren und loslegen.
- **Aktuell**: Stand 2025 vollständig funktionsfähig und mit der aktuellen GitHub‑Oberfläche getestet.

### Erste Schritte

Zum Start benötigen Sie:

1. Die Tampermonkey‑Erweiterung in Ihrem Browser.
2. Das Script selbst, verfügbar auf der [GitHub‑Projektseite](https://github.com/isyuricunha/auto-github-follow-and-unfollow).
3. Folgen Sie den Installationshinweisen in der README des Repositories.

Schauen Sie sich den Code an, tragen Sie bei oder melden Sie Probleme direkt im [GitHub‑Repository](https://github.com/isyuricunha/auto-github-follow-and-unfollow), um das Tool weiter zu verbessern!

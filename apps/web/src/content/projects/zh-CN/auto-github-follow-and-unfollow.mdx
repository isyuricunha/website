## 关于项目

**Auto GitHub Follow and Unfollow（GitHub 自动关注/取消关注）** 是一款轻量级用户脚本，旨在简化 GitHub 连接的管理。无论是希望扩展人脉还是清理关注列表，该脚本都能自动化从 GitHub 个人主页或关注者/正在关注页面进行关注和取消关注的操作。

### 功能介绍

- **自动化关注**：快速关注多个用户，无需手动逐个点击个人主页。
- **批量取消关注**：轻松批量取消关注用户，节省管理连接的时间。
- **用户友好**：无缝集成到 GitHub 的个人主页和关注者/正在关注列表中，融入浏览体验。

### 工作原理

该脚本基于 JavaScript 并通过 Tampermonkey 运行，Tampermonkey 是一款流行的用户脚本浏览器扩展。安装后，它会为 GitHub 界面增加功能，让您能以最小的操作完成关注和取消关注的动作。它兼容基于 Chromium 的浏览器（如 Chrome、Edge、Brave 和 Opera）和 Firefox，确保广泛的可访问性。

### 为什么使用它？

- **高效**：非常适合管理大型网络或经常与多个 GitHub 个人主页互动的用户。
- **简便**：无需复杂设置——只需通过 Tampermonkey 安装脚本即可开始使用。
- **持续更新**：截至 2025 年，仍完全可用，并经过测试，可与 GitHub 最新界面兼容。

### 开始使用

要使用该脚本，您需要：

1. 在浏览器中安装 Tampermonkey 扩展。
2. 脚本本身，可在[项目的 GitHub 页面](https://github.com/isyuricunha/auto-github-follow-and-unfollow)获取。
3. 按照仓库中的 README 文件中的安装说明进行设置。
